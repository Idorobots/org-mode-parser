** Protocol message format

The wire format for a =PARSE_REQUEST= message is as follows:

-----

: Offset  Size  Field
: ------  ----  -----
:      0     4  Magic number: 0x4F524721  ("ORG!")
:      4     2  Version: 0x0002
:      6     2  Flags (bitfield)
:      8     4  Payload length (uint32, big-endian)
:     12     N  Payload (UTF-8 encoded Org source)

-----

Flags field (2 bytes):

: Bit 15: strict_mode     — fail on unrecognised syntax
: Bit 14: include_src     — parse source block contents
: Bit 13: resolve_links   — resolve fuzzy links at parse time
: Bits 12-0: reserved, must be zero

-----

Response format:

: Offset  Size  Field
:      0     4  Magic: 0x4F524752  ("ORGR")
:      4     4  Status code (0 = success)
:      8     4  Node count
:     12     4  Result length
:     16     N  Result payload (JSON or msgpack)
