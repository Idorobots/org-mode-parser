* Caption Without an Affiliated Element

This paragraph demonstrates that #+CAPTION: followed by a blank line
becomes a regular keyword, not an affiliated keyword.

#+CAPTION: This is NOT affiliated — a blank line follows.

The paragraph above is the element following the blank line, so the
#+CAPTION: above is parsed as an ordinary (unrecognised) keyword in the
target syntax, not as an affiliated keyword of this paragraph.

Compare with a properly affiliated caption:

#+CAPTION: This IS affiliated — no blank line before the table.
| Properly affiliated table | Column 2 |
|---------------------------+----------|
| Row                       | Data     |
